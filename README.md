# myDiRT-api

## Getting Started

First, install dependencies.

```bash 
cpanm --installdeps .
```

Run the development webserver

```bash
morbo script/my_app
```

Open http://localhost:3000/api/health with your browser to see the result.

API routes can be accessed on http://localhost:3000/api
